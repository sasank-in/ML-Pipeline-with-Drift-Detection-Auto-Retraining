WORKFLOW
================================================================================

1. Setup (First Time)
   run: setup.bat
   creates: venv/, installs packages

2. Start Services
   run: run_all_services.bat
   opens: 5 windows (one per service)

3. Run Demo
   run: python demo.py
   uses: data/lung_disease.csv
   creates: models/, logs entries

4. View Dashboard
   open: http://localhost:8050
   shows: metrics, drift status

5. Stop Services
   run: stop_all_services.bat
   or: close all windows
